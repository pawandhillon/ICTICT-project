== Description ==

Sb login widget that allows a user to login, register, reset their password, see recent activity, time, post and comment count, author info, in one place. You can also customize wordpress mail from name and address.

= Features = 

*	A login and registration widget for your blog
*	Tabs/links for logged out users include login, register, and 'lost password'.
*	Captcha functionality included.
*	Logged in users can see their name, avatar, logged in date, and comments/posts since sign in.
*	User can see his/her recent viewed posts and recent comments made by him/her.
*	User can see his ID, First joined date, total number of his/her posts, total number of comments made by him/her, user level, posts feed, comments feed.
*	AJAX validation makes this thing run smooth
*	Shortcode & PHP Code if you don't want to use widget
*	Options page for customizing the plugin
*	User can add extra link
*	This plugin is so light
*	Smart & nice looking. You love it.
*	Nice warning & success system.
*	You can translate this plugin.

== Installation ==

= Automatic =

* In the admin panel plugins page click Add New
* Search for SB Login 
* Find SB Login in the list an click Install Now
* Click OK when prompted
* Enjoy!

= Manual =

* Extract sb-login.zip to your wp-content/plugins directory.
* In the admin panel under plugins activate SB Login.
* Enjoy!
